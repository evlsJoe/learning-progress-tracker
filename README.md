# Learning Progress Tracker

A comprehensive progress tracking application for three learning methods: Chess, Coding, and Math. Built with custom learning psychology in mind.

## Features

- **📊 Dashboard**: Overview of progress across all three subjects
- **♟️ Chess Tracker**: Track position-first learning with reconstruction metrics
- **💻 Coding Tracker**: Monitor bug fixes and concept mastery through error-driven learning
- **📐 Math Tracker**: Visualize formulas and problem-solving progress
- **📈 Analytics**: Charts, heatmaps, and streak tracking
- **💾 Data Export/Import**: Save and load your progress data as JSON
- **🌙 Dark Theme**: Professional dark mode interface

## Learning Methods

### Chess: Position-First Learning
Track your board state reconstructions and visual learning progress.

### Coding: Cause → Effect Memory
Learn through debugging and error signatures. Track bugs fixed and concepts mastered.

### Math: Compression + Visualization
Learn formulas as objects with visual transformations. Track rebuilds and visualizations.

## How to Use

1. Open `index.html` in your browser
2. Select a topic (Chess, Coding, or Math)
3. Log your sessions with metrics like:
   - Duration
   - Focus level
   - Energy level
   - Difficulty
   - Topic-specific metrics
4. View progress on the Dashboard
5. Export your data anytime

## Technology

- Pure HTML5
- CSS3 with dark theme
- Vanilla JavaScript
- Chart.js for visualizations
- LocalStorage for data persistence

## Data

All data is stored locally in your browser. Use the Export button to download your data as JSON.
